# Blog Writing Template

## Overview

A comprehensive template for creating engaging blog posts using A-PIRO methodology. This template ensures your content is well-structured, SEO-optimized, and audience-focused.

## When to Use

- Writing technical blog posts
- Creating thought leadership content
- Developing tutorial content
- Writing industry analysis pieces

## Template

```markdown
**Phase 1: Intent Recognition**
- **Objective**: Write [BLOG_TYPE] blog post about [TOPIC]
- **Target Audience**: [DEMOGRAPHIC/EXPERTISE_LEVEL]
- **Success Criteria**:
  - [X minutes read time]
  - [Engagement metrics]
  - [SEO ranking targets]
- **Distribution**: [PLATFORMS where content will be published]

**Phase 2: Evidence-Based Analysis**
- **Topic Research**:
  - Current trends in [INDUSTRY]
  - Pain points of [AUDIENCE]
  - Competitor analysis: [KEY_COMPETITORS]
- **SEO Keywords**: [PRIMARY, SECONDARY keywords]
- **Content Gaps**: [Identified gaps in existing content]

**Phase 3: Structured Reasoning**

**Content Flow Architecture**:
1. **Hook** (First 100 words)
   - Problem statement
   - Surprising statistic
   - Provocative question

2. **Context** (Next 200 words)
   - Background information
   - Why this matters now
   - Personal connection

3. **Core Content** (Main body)
   - [Point 1 with evidence]
   - [Point 2 with examples]
   - [Point 3 with case studies]

4. **Application** (Practical takeaways)
   - Action steps
   - Implementation guide
   - Tool recommendations

5. **Conclusion** (Final 150 words)
   - Summary of key points
   - Call to action
   - Future outlook

**Phase 4: Cross-Reference Validation**
- SEO keywords naturally integrated
- Internal links to related content
- External sources properly cited
- Brand messaging consistent

**Phase 5: Information Architecture**

**Blog Post Structure**:
```markdown
# [Compelling, SEO-Friendly Title]

*[Subtitle that hooks the reader]*

## Why This Matters [Brief intro explaining relevance]

## [Section 1: Problem/Challenge]
### The Current Situation
### What's at Stake

## [Section 2: Solution/Approach]
### Step-by-Step Process
### Real-World Examples

## [Section 3: Implementation]
### Getting Started
### Common Pitfalls to Avoid

## [Section 4: Results/Impact]
### Metrics and Outcomes
### Success Stories

## Key Takeaways
- [Bullet point 1]
- [Bullet point 2]
- [Bullet point 3]

## Next Steps
[Call to action]

---
*About the Author: [Brief bio with link to social/profile]*
```

**Phase 6: Intent Clarification**
- **Tone**: [Professional/Casual/Technical/Inspirational]
- **Format**: List-based/Narrative/How-to/Opinion
- **Length**: [Target word count]
- **Visual Elements**: [Images, charts, videos needed]

- **SEO Requirements**:
  - Meta description (150-160 characters)
  - Alt text for images

- **Email Newsletter**: Integrated promotional content

- **Readability Score**: Target [Flesch-Kincaid level]
- **Expertise Level**: [Beginner/Intermediate/Advanced]
- **Content Freshness**: [Time-sensitive content check]

- **Content Strategy Agent**: Audience alignment and positioning

**Phase 10: RoT Memory Storage**

**Implementation Framework**:

**Title Optimization**:
```markdown
Primary Formula: [Number] + [Adjective] + [Topic] + [Benefit]
Examples:
- "7 Proven Ways to Boost Your Blog Traffic"
- "The Ultimate Guide to Remote Work Productivity"
- "Why 90% of Startups Fail (And How to Be in the 10%)"

SEO Elements:
- Primary keyword in title
- Power words (Ultimate, Complete, Guide, Proven)
- Numbers and specifics
- Emotional triggers
```

**Introduction Formula**:
```markdown
1. **Hook**: Start with statistic, question, or bold statement
2. **Problem**: Explain the pain point
3. **Solution Preview**: Hint at what they'll learn
4. **Credibility**: Why they should trust you
5. **Promise**: What they'll gain from reading
```

**Content Blocks**:
```markdown
Each section should include:
- **Main Point**: Clear topic sentence
- **Evidence**: Data, examples, case studies
- **Explanation**: How to apply
- **Transition**: Smooth connection to next point
```

**Conclusion Structure**:
```markdown
1. **Recap**: 2-3 sentence summary
2. **Main Insight**: The key takeaway
3. **Call to Action**: Specific next step
4. **Engagement Question**: Encourage comments
5. **Teaser**: Hint at future content
```

**SEO Checklist**:
- [ ] Primary keyword in URL
- [ ] Primary keyword in first paragraph
- [ ] Meta description optimized
- [ ] Header tags (H1, H2, H3) used correctly
- [ ] Internal links (2-3 minimum)
- [ ] External links to authoritative sources
- [ ] Image alt text optimized
- [ ] Readability score above 60
- [ ] Mobile-friendly formatting
- [ ] Call to action included

**Promotion Strategy**:
```markdown
Day 1: Email newsletter
Day 2: Twitter thread
Day 3: LinkedIn post
Day 4: Reddit relevant communities
Day 5: Facebook groups
Day 6: Instagram carousel
Day 7: Repurpose to video script
```

**Success Metrics**:
- **Traffic**: Page views, unique visitors
- **Engagement**: Time on page, bounce rate, comments
- **Conversion**: Email sign-ups, resource downloads
- **Social**: Shares, mentions, backlinks
- **SEO**: Keyword rankings, organic traffic growth
```

- **Tell stories**: Personal anecdotes increase engagement by 22%
- **Optimize for featured snippets**: Answer common questions directly
